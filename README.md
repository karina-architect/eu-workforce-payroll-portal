# EuroWorkforce Enterprise

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Upload the folder to Vercel or run:
```bash
npm run build
npm run start
```

## Included
- Full deployable Next.js project
- Enterprise UI
- Conversion-optimized funnel
- Lead capture section
- Multi-language experience
- SEO/discovery layer
- Consultant persuasion engine
